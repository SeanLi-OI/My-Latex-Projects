MCMthesis for MCM Version 5.0y
==============================
第二版说明：（2018.2.8）
2018美赛马上到来，最近有许多网友反映模板使用过程中的一些问题，特对模板作了更大程度的修改，主要有如下几点：
-删除了原来模板中所有不合美赛规范的内容（代码部分，附录，sheet和abstract的可选性）
-注释全部汉化，并补加了大量注释以方便小白学习使用（在MCMthesis.sty文件中）
-改进了第一版中一些不规范的部分（sheet页顶部边距过大，sheet页有页码等）
祝大家美赛顺利，取得好成绩！

第一版说明：（2018.1.18）
原模板（5.0）来自LaTeX工作室（latexstudio.net），经youjiarui189按照比赛规范做了一点小小的修改（5.0y）。

文件：
-XXXXX.tex：主文件，包括了基本信息和摘要部分（使用时请自行修改文件名为你们组的控制号）
-body.tex：正文部分的文件
-MCMthesis.sty：模板宏包文件，默认为MCM，若需要改为ICM标题请到该文件末尾处的一个center环境中修改相关内容
